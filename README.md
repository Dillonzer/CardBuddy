# Slash Commands
- `/card`: Displays a picture of the card in that set or all cards with that name!
- `/what_set_is_this_card_in`: PMs you a list of all the sets that card is in!
- `/card_text`: Displays a picture of the card that contains the text you are searching!
- `/all_sets`: PMs you a list of all sets with set codes!
- `/set`: Prints out information about the set you are asking about!
- `/set_checklist`: DMs you a Checklist for the set you asked for
- `/set_list`: Displays all card in that set! (TCGPlayer Integration disabled for this command)
## PokemonCards.io Commands
- `/get_deck`: Get deck information from PokemonCards.io!
- `/get_random_deck`: Get a random deck from PokemonCards.io!
## Admin Commands
- `/tcgplayer_toggle`: Toggles the TCGPlayer Integration!
- `/ptcgl_crafting_toggle`: Toggles the PTCGL Crafting Price Functionality!

# Tips & Tricks
Here are some useful tips to utilze the bot:
- Only the original requester of the command can utilize the :arrow_backward: and :arrow_forward: to cycle the cards.
- V, VMAX and VSTAR cards are denoted as `<cardName> V`, `<cardName> VMAX`, and `<cardName> VSTAR`
- GX and EX cards are denoted as `<cardName> GX` and `<cardName> EX`
- Prism Star cards are denoted as `<cardName> {*}`
- BREAK cards are denoted as `<cardName> BREAK`
- LEGEND cards are denoted as `<cardName> LEGEND`
- Mega cards are denoted as `M <cardName> EX`
- G / GL cards are denoted as `<cardName> G`
- Elite 4 cards are denoted as `<cardName> E4`
- C / CL cards are denoted as `<cardName> C`
- Primal cards are denoted as `Primal <cardName> EX`
- Delta Species cards are denoted as `<cardName> Delta Species`
- Dark cards are denoted as `Dark <cardName>`
- Light cards are denoted as `Light <cardName>`
- Rocket's cards are denoted as `Rocket's <cardName>`
- Prime cards have no special modifiers in their name. Feel free to just use `<cardName>`
- Some sprites will not show up. This is normal. It will only show sprites it can find.

# FAQ
### Q: When do new sets get added?
A: New sets are uploaded to the API usually the day before or on the Street Release Day.
